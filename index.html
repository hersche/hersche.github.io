jobManagement

For undeprecated information please consult README.md on Github

This is a programm for temporary workers. It's specialiced for diffrent companys, so when u like to switch, it may could help you. The target is to watch the companys and how much money they give you (cause a company isn't trustable). The programm supports:

  • Companys
  • Jobs
  • Weekends
  • Month-over-jobs
  • Charges
  • Credits
  • Diffrent filtering options and Views
  • Overallview and Detailedview based on Companys
  • Calendarview
  • Livesearch [jobname, jobplace, jobleader, jobcomment, companyname]
  • Encryption by pycrypto

Peta

This PHP-application was based on v5.2 and continues with 5.5 now. Primary goal was one simple application, but it was used as base for many kinds of apps by myself. The name Peta came out of my idea called PHPMeta. This because the goal goes now in a very modular direction. The old name "learncards-web" is death!

  • MVC-arch
  • Dojo for JS
  • Smarty for templates
  • Users with profiles and roles
  • plugins with roles and rights
